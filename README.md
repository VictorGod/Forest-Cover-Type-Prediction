# Forest-Cover-Type-Prediction
Представленный код выполняет несколько стандартных задач, связанных с обработкой данных, обучением моделей и настройкой гиперпараметров. Код использует популярные библиотеки pandas и scikit-learn, что является распространенной практикой в анализе данных и машинном обучении.
Более точно он выполняет следующие задачи для демонстрации навыков с представленнымими библиотеками:
1.Импортирует необходимые библиотеки: pandas, numpy, sklearn.
2.Считывает CSV-файл ('forest_dataset.csv') в объект DataFrame pandas под названием all_data.
3.Выводит первые несколько строк DataFrame с помощью метода head().
4.Выводит форму DataFrame с помощью атрибута shape.
5.Извлекает метки из последнего столбца DataFrame и присваивает их переменной labels.
6.Извлекает матрицу признаков из всех столбцов, кроме последнего, и присваивает ее переменной feature_matrix.
7.Разделяет данные на обучающую и тестовую выборки с помощью функции train_test_split() из sklearn.
8.Создает модель логистической регрессии с указанным гиперпараметром C и обучает ее на обучающих данных.
9.Делает предсказания на тестовых данных с использованием обученной модели.
10.Вычисляет показатель точности модели, сравнивая предсказанные метки с фактическими метками.
11.Выполняет поиск по сетке для нахождения лучших гиперпараметров для модели логистической регрессии.
12.Выводит лучшие найденные параметры поиска по сетке.
13.Вычисляет показатель точности модели с лучшими параметрами на всем наборе данных.
14.Создает классификатор k-ближайших соседей с указанными гиперпараметрами и обучает его на обучающих данных.
15.Делает предсказания на тестовых данных с использованием обученного классификатора k-ближайших соседей.
16.Вычисляет показатель точности классификатора k-ближайших соседей.
17.Выполняет поиск по сетке для нахождения лучших гиперпараметров для классификатора k-ближайших соседей.
18.Выводит лучшие найденные параметры поиска по сетке.
19.Создает оптимальный классификатор k-ближайших соседей с лучшими параметрами и обучает его на обучающих данных.
20.Делает вероятностные предсказания на тестовых данных с использованием оптимального классификатора.
21.Вычисляет среднюю вероятность класса 2 (индекс 2) из предсказанных вероятностей.
22.Строит столбчатую диаграмму, сравнивающую предсказанные частоты классов с фактическими частотами классов.
